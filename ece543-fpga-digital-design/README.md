# ECE 543 — Digital Design & FPGA Systems

University of New Hampshire, ECE Department
Instructor: Dr. Sandeep Sunkavilli

## Course Overview

This course covers digital logic design and FPGA-based system implementation, with
labs alternating weekly between breadboard-level circuits and FPGA implementation
on the Cora Z7-07S platform.

## Structure

- `labs/` — weekly lab assignments, alternating breadboard and FPGA weeks
- `reference-designs/` — instructor reference implementations (I2C/AXI block design, pin mapping)
- `capstone-starters/` — starter scaffolding for capstone projects

## Hardware

- **FPGA Platform:** Digilent Cora Z7-07S (12 units)
- **Toolchain:** Xilinx Vivado / Vitis

## Capstone Tracks

1. **Biosignal Anomaly Detection** — FPGA-based biosignal processing with PL vs. PS
   baseline performance comparison
2. **Triage FSM (NEWS2)** — Verilog finite state machine implementing NEWS2 clinical
   triage scoring logic
