# ECE 543 — Digital Design & FPGA Systems

University of New Hampshire, ECE Department, Instructor: Dr. Sandeep Sunkavilli

## Course Overview

This course covers digital logic design and FPGA-based system implementation, with labs alternating weekly between breadboard-level circuits and FPGA implementation on the Xilinx BASYS 3 and Cora Z7-07S platform.

## Structure

- Theory classes Monday, Wednesday and Friday in-person in UNH Durham
- `labs/` — weekly lab assignments, alternating breadboard (TTL) and FPGA weeks
- `reference-designs/` — instructor reference implementations

## Hardware

- **FPGA Platform:** Xilinx BASYS 3 and Digilent Cora Z7-07S (12 units)
- **Toolchain:** Multisim and Xilinx Vivado / Vitis
