# Capstone: Triage FSM (NEWS2 Scoring)

## Objective
Implement a Verilog finite state machine that computes NEWS2
(National Early Warning Score 2) clinical triage scores from vital sign inputs.

## Inputs
- Respiratory rate
- SpO2
- Systolic blood pressure
- Pulse rate
- Consciousness level
- Temperature
- Supplemental oxygen (yes/no)

## Deliverables
- Verilog FSM implementation
- Testbench with sample patient vitals
- Score verification against NEWS2 reference table
