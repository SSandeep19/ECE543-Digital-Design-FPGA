# Capstone: Biosignal Anomaly Detection (FPGA)

## Objective
Implement biosignal processing (PPG/ECG) on FPGA fabric (PL) and compare
performance against a processor-based (PS) software baseline.

## Core Research Claim
PL implementation should demonstrate lower latency and/or lower power
consumption versus equivalent PS implementation.

## Suggested Milestones
1. PS baseline implementation (software FIR filters)
2. PL implementation (HLS or RTL FIR filters)
3. Benchmark comparison: latency, throughput, power
4. Report / poster
